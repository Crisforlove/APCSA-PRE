#### 2
Given `int[] arr = {3, 4, 5, 6};`, which expression accesses the **last** element?

- A) arr[22]
- B) arr[arr.length]
- C) arr[arr.length - 1]
- D) arr.last()
- E) arr.get(3)

Answer: C

#### 4
What is the result of executing `arr[arr.length] = 7;`?

- A) The code will fail to compile.
- B) A runtime ArrayIndexOutOfBoundsException is thrown.
- C) The array will automatically expand.
- D) The assignment is ignored.
- E) The first element is overwritten.

#### 5
For an array arr, which is the correct way to retrieve the number of elements?

- A) arr.size()
- B) arr.length()
- C) arr.length
- D) arr.capacity()
- E) arr.getLength()

Answer: C

#### 6
Consider the following code segment:
```java
int[] a = {1, 2, 3};
for (int x : a) {
    x = x * 2;
}
```
What is a after execution?

- A) {2, 4, 6}
- B) {1, 2, 3}
- C) {1, 4, 9}
- D) {2, 2, 2}
- E) A compiler error occurs.

Answer: B

#### 7
Consider the following code segment:
```java
int[] a = {1, 2, 3};
int[] b = a;
b = 9;
```
What is a?

- A) 1
- B) 2
- C) 3
- D) 9
- E) It is undefined.

Answer: D
