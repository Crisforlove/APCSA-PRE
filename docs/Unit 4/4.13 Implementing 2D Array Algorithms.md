```java

for (int[] row : arr) {
for (int value : row) {
if (value > 0) {
ans = ans * value;
}
}
}
```

- A) It prints the product of all elements in arr.
- B) It prints the product of all negative elements in arr.
- C) It prints the product of all positive elements in arr.
- D) It prints nothing because of a run-time error if zero is present.
