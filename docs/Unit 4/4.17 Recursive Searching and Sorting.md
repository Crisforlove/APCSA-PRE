### Topic 4.17 — Recursive Searching and Sorting

#### 1
Which code correctly completes method power to handle negative exponents where $a^{-n} = 1/a^n$?

- A) (1.0 / base) * power(base, expo + 1)
- B) (1 / base) * power(base, expo - 1)
- C) base * power(base, expo + 1)
- D) base * power(base, expo - 1)
- E) (1 / base) * power(base, expo)

Answer: A

#### 2

- A) 3211211
- B) 1121213
- C) 1213121
- D) 1211213
- E) 1123211

Answer: C

#### 3
Which code satisfies the postcondition of writeEven (printing even integers in reverse order of input)?

- D) I and II only
