#### 1
Consider the following code:
```java
Person a = new Person("Amy");
Person b = a;
b.setName("Bo");
```

- A) Amy
- B) Bo

#### 2
What happens after this call?
```java
public void reset(Person p) { p = new Person("New"); }
// call:

```

- B) p1 becomes null.

- D) A runtime error occurs.

```java

// call:
update(p1);
```

- B) p1 now refers to a different object.

#### 4
