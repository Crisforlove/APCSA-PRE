Topic 1.6 — Compound Assignment Operators

1

Consider the following code segment.

int x = 7;
int y = 3;
x += y;
y *= 2;
x %= y;

What is the value of x after the code segment has been executed?

  • A) 1
  • B) 4
  • C) 10
  • D) 6
  • E) 0

Answer: B

2

Which of the following is equivalent to the statement x \= x + 1;?

  • I. x++;

  • II. x += 1;

  • III. ++x; (Note: While valid Java, CED focuses on post-increment)

  • A) I only

  • B) II only
  • C) I and II only
  • D) II and III only
  • E) I, II, and III

Answer: E

3

Consider the following code segment.

int result = 20;
result /= 4;
result += 2;
result *= 3;

What is the value of result?

  • A) 15
  • B) 21
  • C) 11
  • D) 7
  • E) 18

Answer: B

4

Consider the following code segment.

int k = 10;
k++;
k--;
k += 5;
k -= 2;

What is the final value of k?

  • A) 10
  • B) 15
  • C) 13
  • D) 12
  • E) 14

Answer: C

5

A programmer wants to double the value of a variable score. Which of the following statements will accomplish this?

  • I. score \= score * 2;

  • II. score *= 2;

  • III. score += score;

  • A) I only

  • B) II only
  • C) I and II only
  • D) II and III only
  • E) I, II, and III

Answer: E

6

What is the value of n after the following code executes?

int n = 15;
n %= 4;
n += n;
  • A) 3
  • B) 6
  • C) 7
  • D) 1
  • E) 2

Answer: B

7

Consider the following code segment.

int a = 5;
int b = 2;
a *= b + 3;

What is the value of a?

  • A) 13
  • B) 25
  • C) 10
  • D) 15
  • E) 11

Answer: B

8

Which compound assignment operator is used to find the remainder and assign it to the variable?

  • A) rem=
  • B) %=
  • C) /=
  • D) ^=
  • E) &=

Answer: B

9

Consider the following code segment.

int count = 0;
count++;
count++;
count--;

What is the value of count?

  • A) 0
  • B) 1
  • C) 2
  • D) -1
  • E) 3

Answer: B

10

Which of the following statements about compound assignment operators is true?

  • A) They can only be used with int variables.
  • B) They perform an operation and an assignment in one step.
  • C) x += 1 is slower than x = x + 1.
  • D) They cannot be used with double variables.
  • E) They are ignored by the compiler.

Answer: B
